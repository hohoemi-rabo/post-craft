## コンポーネント仕様

### Props

```typescript
interface EmojiPickerProps {
  value: string           // 現在選択中の絵文字
  onChange: (emoji: string) => void
}
```

### UI

- ボタンクリックでポップオーバー表示
- カテゴリタブで切り替え
- グリッドレイアウトで絵文字を表示
- 選択中の絵文字をハイライト
- 外クリックで閉じる

### サイズ

- タッチターゲット: 最低 44x44px
- ポップオーバー幅: 280px 程度

## 受入条件

- カテゴリ切り替えが動作する
- 絵文字選択が `onChange` で親に伝わる
- 選択中の絵文字がハイライト表示される
- ポップオーバーの外クリックで閉じる
- モバイルでも使いやすいサイズ
- ダークテーマに対応
