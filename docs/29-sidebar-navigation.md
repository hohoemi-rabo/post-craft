# チケット #29: サイドバー更新（設定サブナビゲーション）

> Phase 3 | 優先度: 高 | 依存: なし

## 概要

サイドバーとモバイルナビの「設定」項目にサブナビゲーション（投稿タイプ、ハッシュタグ）を追加する。設定トップページにもリンクカードを追加する。

SPEC-PHASE3.md セクション 8.2 に準拠。

## 対象ファイル

| ファイル | 操作 |
|---------|------|
| `src/components/dashboard/sidebar.tsx` | 更新 |
| `src/components/dashboard/mobile-nav.tsx` | 更新（存在すれば） |
| `src/app/(dashboard)/settings/page.tsx` | 更新（リンクカード追加） |

## サイドバーの変更

```
📊 ダッシュボード
📝 投稿作成
📚 履歴
👤 キャラクター
⚙️ 設定
   └─ 📝 投稿タイプ     ← 新規追加
   └─ #️⃣ ハッシュタグ   ← 新規追加
```

- 設定ページ（`/settings/*`）表示時にサブメニューを展開
- 現在のパスに応じてアクティブ状態をハイライト

## 設定トップページの変更

既存のアカウント情報に加えて、以下のリンクカードを追加:
- 📝 投稿タイプ管理 → `/settings/post-types`
- #️⃣ ハッシュタグ設定 → `/settings/hashtags`

## 受入条件

- サイドバーに設定サブメニューが表示される
- 設定配下のページにいる時、サブメニューが展開されている
- 現在のページがハイライトされる
- モバイルでもサブページへの導線がある
- 設定トップにリンクカードが表示される
- 既存の設定ページ（アカウント情報・ログアウト）が壊れていない
